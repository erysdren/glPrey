glPrey v1.0.0
=========================

A simple OpenGL-enabled viewer for Prey (1995) BSP files using SDL2.

SDL2.dll is included here, but can be updated for any future version of SDL2.

Requires MACT.WAD and DEMO4.BSP from the Prey 1995 alpha to run.

Those can be found here:

https://archive.org/details/prey-1195/
